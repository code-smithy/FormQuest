# Fitness RPG MVP — Project Management Docs

This folder contains execution docs for planning, sequencing, and tracking delivery of the Fitness RPG MVP.

## Documents
- `work-breakdown-structure.md` — full WBS by stream and deliverable.
- `milestones.md` — milestone definitions, dates, and exit criteria.
- `mvp-backlog.md` — prioritized backlog with acceptance criteria.
- `progress-tracker.md` — status dashboard and weekly reporting template.
- `risks-and-dependencies.md` — active risks, owners, mitigations, and external dependencies.

## Suggested usage
1. Review `work-breakdown-structure.md` and confirm ownership.
2. Use `mvp-backlog.md` to pull work into weekly sprint planning.
3. Update `progress-tracker.md` at least twice per week.
4. Reassess `risks-and-dependencies.md` weekly.
5. Validate milestone readiness against `milestones.md` exit criteria.
